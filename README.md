# finance-miniprogram

微信小程序初始化项目（理财方向）。

## 已包含内容

- 基础小程序入口：`app.js` `app.json` `app.wxss`
- 首页：`pages/index/*`
- 项目配置：`project.config.json` `sitemap.json`
- Git 忽略规则：`.gitignore`

## 本地运行

1. 打开微信开发者工具
2. 导入当前目录
3. `AppID` 可先使用测试号，后续替换 `project.config.json` 中的 `appid`
